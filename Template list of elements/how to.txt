Чтобы распечатать лист регистрации изменений для схемы Э3, нужно:
- открыть файл KPTV_LRI
- нажатием клавиши "S" открыть окно редактирования основной надписи
- вписать нужный номер вида КПТВ.??????.??? Э3
- перейти на 2 страницу
- нажать в верхнем меню клавишу с изображением карандаша, номер страницы должен стать розовым
- двойным кликом по номеру страницы открыть поле редактирования текста, вписать нужный номер листа
- распечатать только вторую страницу

Я не нашел способа оставить в документе TDD только лист регистрации изменений, поэтому приходится не забывать не печатать первый лист.
